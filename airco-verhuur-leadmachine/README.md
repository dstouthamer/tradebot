# Leadmachine — Mobiele Airco Verhuur

Compleet draaiboek voor een WordPress-website + leadmachine die hoog rankt in Google, veel leads genereert, leads razendsnel opvolgt en aanvragen omzet naar verhuurklanten.

Gebouwd als gesimuleerd team: **projectmanager, WordPress-developer, SEO-specialist, copywriter, CRO-specialist, automation-specialist en sales closer.**

## De kernlogica van de leadmachine

```
GOOGLE (SEO + Ads)  ─┐
SOCIAL / DOORVERWIJZING ─┼──►  LANDINGSPAGINA  ──►  LEADFORMULIER / WHATSAPP / BELLEN
                     ─┘            (CRO)                        │
                                                                ▼
                                                  CRM-LEAD (binnen <5 min reactie)
                                                                │
                          ┌──── AUTOMATISCHE OPVOLGING ─────────┤
                          │   (WhatsApp + e-mail + bel-taak)    │
                          ▼                                     ▼
                    OFFERTE / VOORSTEL  ──────────►  SALES CLOSE  ──►  VERHUURKLANT
                          │                                              │
                          └──────────── REVIEW + REPEAT/SEIZOEN ◄────────┘
```

**Het verschil tussen winnen en verliezen in deze markt = reactiesnelheid.** Mobiele airco's worden gehuurd op hete dagen, vaak met spoed. Wie binnen 5 minuten belt/appt, wint de deal. De hele machine is daarop gebouwd.

## Snelstart (eerste 30 dagen)

1. **Week 1** — Domein + hosting + WordPress + thema (zie 11). Tracking opzetten (zie 12).
2. **Week 2** — Homepage + 3 dienstpagina's + leadformulier live (zie 02, 04, 06).
3. **Week 3** — 5 lokale SEO-pagina's + Google Business Profile + CRM-pipeline (zie 05, 09).
4. **Week 4** — Google Ads aanzetten op piekdagen + opvolg-automatisering testen (zie 10, 08).

Daarna: doorlopende SEO-content + CRO-optimalisatie (zie 03, 12).
